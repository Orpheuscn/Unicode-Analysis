# Unicode Analysis

这是一个 Unicode 字符分析和展示项目。

## 功能特性

- Unicode 字符数据展示
- 字符属性分析
- 交互式界面

## 在线访问

访问 GitHub Pages: [https://orpheuscn.github.io/Unicode-Analysis/](https://orpheuscn.github.io/Unicode-Analysis/)

## 文件说明

- `index.html` - 主页面
- `unicode_data.json` - Unicode 字符数据

## 部署

本项目使用 GitHub Actions 自动部署到 GitHub Pages。
